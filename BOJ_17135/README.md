# [BOJ 17135. 캐슬 디펜스](https://www.acmicpc.net/problem/17135)


## How to solve

* 시뮬레이션 문제
* prev_permutation 함수를 통해 아처의 배치 모든 경우의 수를 만듦
* 커스텀 cmp를 이용해 거리가 가장 짧고, X좌표가 가장 짧은 것을 우선순위로 정렬하도록 하여, 각 배열의 가장 첫번째 값이 저격당할 병사로 정함.
