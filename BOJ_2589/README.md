# [BOJ 2589. 보물섬](https://www.acmicpc.net/problem/2589)


## How to solve
* BFS로 풀이
* 모든 육지 지점에서 각각 갈 수 있는 최단거리중 가장 긴 경로를 찾아 result로 반환
* queue의 남용으로 메모리 초과를 초례하였음.
* x, y좌표 순서 잘 신경써야함. 이로 인해 문제가 계속 틀렸음.


## Build

$ make
